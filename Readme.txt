This code accompany thesis : Full-Field Analysis : A Machine Learning Approach by Abdullah Alakeely.\
\
Files:\
*****run_file: use this to run the code, adjust parameters as necessary,  the code will generate the following:\
	1- A plot of prediction (in color) and actual curves (gray) for the surfac measurment problem.\
	2- Create a folder with name you choose in varible \'93results_directory\'94. Default = Experiment1.\
	3- In the folder, 3 other folders to record plots, parameter and score files , and prediction files.\
	4- A print will be displayed on the screen showing the results for gas, oil and water curves.\
\
\
*****supporting_modules: contain class for Models, DataProcessing, and recording using Pandas data frame.\
